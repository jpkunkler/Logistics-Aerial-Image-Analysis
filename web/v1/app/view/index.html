Klassifizierung von Logistikstandorten anhand von Luftbildern

Bitte laden Sie eine Luftbildaufnahme eines deutschen Standortes (Lager, Distributionszentrum, Filiale) hoch. Das Modell klassifiziert den Standort in vier Klassen: Sehr Gut, Gut, Mittel und Schlecht.
Die Klassifizierung beschreibt den Grad der Einfachheit einer Belieferung/Anfahrt.
